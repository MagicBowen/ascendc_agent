# Ascend C Develop Guide

## Online docs

本文档给出了使用 Ascend C 进行开发的官方文档的链接，当你需要了解更多开发细节的时候，使用 WebFetch 或 WebSearch 能力在对应的在线文档中进行浏览查找。

- [Ascend C 算子开发样例详解](https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/83RC1/opdevg/Ascendcopdevg/atlas_ascendc_10_0005.html) : 该文档详细介绍了 Ascend C 算子开发的样例代码，包括样例 add_custom 的目录结构、代码实现、运行方式、关联文档等内容。通过阅读该文档，开发者可以了解如何使用 Ascend C 进行算子开发的基本流程和方法。

- [Ascend C ACL 接口参考](https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/83RC1/API/appdevgapi/aclcppdevg_03_1952.html) : 该文档介绍了 Ascend C 的 Host 主机侧代码开发，需要使用 ACL 接口。参考本文档及其关联文档中的介绍和接口说明，开发者可以使用这些接口来管理内存、创建和操作张量、启动核函数等。

- [Ascend API 手册](https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/83RC1/API/ascendcopapi/atlasascendc_api_07_0003.html) ： Ascend C 的核函数开发是 SPMD 模型的编程方式，开发者需要使用 Ascend C 提供的 API 来实现核函数的计算逻辑。该手册详细介绍了 Ascend C 提供的各类 API 接口，包括张量操作、内存管理、控制流等内容。

- [Ascend C 算子开发： 矢量算子](https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/83RC1/opdevg/Ascendcopdevg/atlas_ascendc_10_10001.html) ： 该文档介绍了使用 Ascend C 开发矢量算子的流程和注意事项，包含各种场景下的矢量算子的设计、内存布局、数据搬运等内容。

- [Ascend C 调试工具](https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/83RC1/devaids/optool/atlasopdev_16_0062.html) : 该文档介绍了 Ascend C 的调试工具 msDebug, 一款面向昇腾设备的算子调试工具，用于调试NPU侧运行的算子程序，为算子开发人员提供调试手段。调试手段包括了读取昇腾设备内存与寄存器、暂停与恢复程序运行状态等。

- [Ascend C 官方文档](https://www.hiascend.com/document/detail/zh/CANNCommunityEdition/83RC1/opdevg/Ascendcopdevg/atlas_ascendc_10_0001.html) ： 官方文档的入口，包含了 Ascend C 的基础介绍、开发环境搭建、算子开发流程、性能优化等内容。可以探索更多关于 Ascend C 的开发细节。
